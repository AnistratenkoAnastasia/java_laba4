# Employee CSV Data Processor

## Обработка и парсинг данных сотрудников из CSV в Java

Лабораторная работа, направленная на отработку навыков работы с коллекциями. Проект реализует объектную модель с использованием встроенных Java коллекций, демонстрирует парсинг данных с помощью библиотеки OpenCSV, оптимизацию памяти через кэширование связанных объектов (`Department`), обработку исключений, а также применение Maven, JUnit 4 и JavaDoc.

## Возможности

-  Парсинг CSV-файлов с кастомным разделителем `;` через библиотеку OpenCSV
-  Автоматическая генерация `ID` для класса `Department`
-  Автоматическое кэширование объектов `Department` (`HashMap`) для предотвращения создания дубликатов
-  Обработка ошибок: отсутствие файла, некорректная длина строки, ошибки формата чисел
-  Полное покрытие кода комментариями JavaDoc
-  Unit-тесты на JUnit 4 для проверки логики парсинга и работы сущностей
-  Автоматическая генерация HTML-документации через Maven

## Требования

- JDK 17+
- Maven (встроенный wrapper `mvnw` или системный Maven)
- Зависимость: OpenCSV (указана в `pom.xml`)
